Error Statistics

To avoid overfitting to the specific time period of the dry run, we calculate error statistics for a smaller window size and slide it across the entire time period. Error values are calculated as the difference between the predicted and actual volumes for the given time period, calculated at every point during the dry run. Below, mean error is calculated and shown as a histogram. The window size can be adjusted to see how error changes over different time periods.
